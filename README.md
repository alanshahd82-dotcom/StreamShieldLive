# StreamShieldLive

## رفع المشروع على GitHub وبناؤه تلقائياً

### الخطوات:

1. **أنشئ مستودعاً جديداً على GitHub**
   - افتح [github.com](https://github.com) من هاتفك
   - اضغط على **+** ثم **New repository**
   - اسمه: `StreamShieldLive`
   - اجعله **Public** ثم اضغط **Create repository**

2. **ارفع الملفات**
   - في صفحة المستودع اضغط **uploading an existing file**
   - ارفع جميع ملفات المجلد هذا

3. **انتظر البناء التلقائي**
   - اذهب إلى تبويب **Actions** في المستودع
   - ستجد عملية بناء جارية تلقائياً
   - انتظر بضع دقائق حتى تنتهي ✅

4. **حمّل ملف APK**
   - انقر على آخر عملية بناء ناجحة
   - في الأسفل ستجد قسم **Artifacts**
   - اضغط على **StreamShieldLive-debug** لتحميل ملف APK

5. **ثبّت التطبيق على هاتفك**
   - فعّل **تثبيت التطبيقات من مصادر غير معروفة** في إعدادات هاتفك
   - افتح ملف APK وثبّته
